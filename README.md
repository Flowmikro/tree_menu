# tree_menu
Проект выводит древовидное меню с помощью templatetags 
___
## Установка
Установить репозиторий
```
git clone https://github.com/Flowmikro/tree_menu.git 
```
Перейти в директорию
```
cd tree_menu
```
Виртуальная среда
```
python -m venv venv
source env/bin/activate
```
Установить пакеты
```
pip install -r requirements.txt
```
